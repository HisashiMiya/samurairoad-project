# SamuraiRoad Project

A walking map project to visualize Japan's historic highways (Gokaido):
Tokaido, Nakasendo, Koshu Kaido, Nikko Kaido, Oshu Kaido.

## Features
- Historic route lines (GeoJSON)
- Toggleable POIs: Stations (宿場) and Ichirizuka (一里塚)
- All coordinates are predefined (no external geocoding)

## Structure
- /pages/index.html
- /pages/data/*.geojson
